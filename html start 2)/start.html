Разметка страницы:


для перечисления используется {li} {/li}, можно переносить на несколько строчек
  1. ol - разметка при помощи перечисления (A, B, C, D / 1, 2, 3, 4)
    примеры использования
    1. один
    2. два
    3. три
      и даже в несколько строчек

    в {ol} можно добавить несколько аргуметнов:
    1. reversed - порядок нумерации наоборот
    2. start = "number" нумерация начнётся с указаного числа
    3. type задаёт тип нумерации
      • 1 для цифр (по умолчанию)
      • a для строчных букв
      • A для заглавных букв
      • i для строчной Римской нумерации
      • I для заглавной Римской нумерации
  2. il - перечисление при помощи символов
    3. примеры использования
    • один
    • два
    • три
      и даже несколько строчек
    в {il} можно добавить несколько аргмуентов:
    1. compact - cписок будет в более компактном стиле
    2. type - атрибут добавляющий маркеры в список
      • circle
      • disc
      • square

Добавление изображения:

для того чтобы добавить изображение используется img {img src="" alt=""}

  • src - URL или название изображения. Этот атрибут является обязательным для элемента
  • alt - этим атрибутом задаётся альтернативное текстовое описание изображения.
    • Примечание: Браузеры не всегда отображают изображение на которое ссылается элемент, и в случае если ссылка или название на картинку ошибочное, то вмесо картинки будет заданный текст:
  • width - размер изображения в пикселях
